Изложите в текстовом файле 5-7 интересных, полезных и важных мыслей/идей, которые вы узнали из Learning Code Should Be Easy to Understand.

1. Необходимо выполнять только одно задание в любой момент времени.
2. Если код трудночитаемый, необходимо понять, не выполняет ли блок кода несколько дел одновременно.
3. Если это так, то разбить код, выделяя необходимые подзадачи.
4. Некоторые подзадачи можно реализовать с помощью отдельных функций.
5. Другие могут быть отделены абзацами внутри одного блока.
