Данная программа предназначена для рассмотрения возможностей отладчика Visual Studio.
Задачи:
  1. Подготовка кода к отладке с помощью анализатора кода интегрированной среды разработки.
  2. Исправление исключений (ошибки времени выполнения).
  3. Как сокращать ошибки путем написания кода для намерения (с помощью Assert).
