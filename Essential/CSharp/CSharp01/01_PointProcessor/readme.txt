Что будет делать программа, если очередная строка в файле пустая?
   - пустой файл?
   - файл не существует?
   - в строке файла буква?
   - в строке файла три координаты?
   - в строке файла одна координата?
   - в строке файла только разделитель?

  1) При обработке пустого файла перейдет к обработке следующего файла.
  2) Если файла не существует перейдет к обработке следующего файла.
  3) Перейдет к обработке следующей строки, когда в строке есть ошибка:
      - буква,
      - три координаты,
      - одна,
      - только разделитель.
  4) Чтобы закончилась обработка строк с консоли, введите пустую строку.
